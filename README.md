# CPMA Public Comps Dashboard

Auto-refreshing Engineering & Construction public company comparable analysis dashboard.

## Live Dashboard

Once set up, your dashboard will be live at:
`https://YOUR_USERNAME.github.io/cpma-dashboard/`

## Setup Instructions (One-Time, ~5 minutes)

### Step 1: Create a GitHub Repository

1. Go to [github.com/new](https://github.com/new)
2. Name the repo: `cpma-dashboard`
3. Set it to **Public** (required for free GitHub Pages)
4. Do NOT initialize with README (we'll push our own files)
5. Click **Create repository**

### Step 2: Push These Files

Open a terminal and run:

```bash
cd /path/to/cpma-dashboard-repo
git init
git add .
git commit -m "Initial CPMA dashboard"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/cpma-dashboard.git
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to your repo on GitHub → **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Set branch to `main` and folder to `/ (root)`
4. Click **Save**
5. Wait ~1 minute, then visit `https://YOUR_USERNAME.github.io/cpma-dashboard/`

### Step 4: Add FMP API Key (Optional but Recommended)

The refresh workflow uses the free demo key by default. For better data coverage:

1. Get a free API key at [financialmodelingprep.com](https://financialmodelingprep.com/developer/docs/)
2. Go to your repo → **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret**
4. Name: `FMP_API_KEY`
5. Value: your API key
6. Click **Add secret**

### Step 5: Verify the Workflow

1. Go to your repo → **Actions** tab
2. You should see "Refresh CPMA Dashboard" workflow
3. Click it → **Run workflow** → **Run workflow** to test manually
4. After ~5 minutes, check that the dashboard data updated

## How It Works

- **GitHub Actions** runs `cpma_data_feed.py` at 11pm ET every weekday
- It skips weekends and US market holidays automatically
- Fresh data is embedded into `index.html` and committed back to the repo
- GitHub Pages serves the updated file — same URL, always fresh data

## Files

| File | Purpose |
|------|---------|
| `index.html` | The dashboard (self-contained, data embedded) |
| `cpma_data_feed.py` | Fetches market data from FMP API |
| `cpma_refresh.py` | Rebuilds the dashboard HTML with new data |
| `cpma_comps_data.json` | Full dataset (generated by data feed) |
| `.github/workflows/refresh.yml` | GitHub Actions workflow for auto-refresh |

## Manual Refresh

To refresh data outside the schedule:
- Go to **Actions** → **Refresh CPMA Dashboard** → **Run workflow**

Or locally:
```bash
python cpma_refresh.py --api-key YOUR_KEY
```
